1. First run the SPIFFS_Initialize.ino, compile it and upload on the ESP32. Open the Serial monitor and reset the ESP32
   and Keep looking at the Serial monitor, when the file read/write test is complete then colse it.

2. Next run Gif_ESP32_GIF_Upload.ino, compile it and upload on the ESP32.

3. Press the B1 button to connect to the WiFi network whose credential you put in the Gif_ESP32_GIF_Upload.ino. Once the 
   ESP32 is connected to the WiFi it will display the Network SSID and the IP address on the screen.

5. Put the IP address in the webbrowser to access the webserver to upload GIF files to ESP32 flash memory.
